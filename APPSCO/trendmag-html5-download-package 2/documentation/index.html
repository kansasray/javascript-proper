Responsive

"5. - Responsive" in file "style.css" contains styling for media queries that allows us to tailor to different resolutions and devices without changing the content. The file is separated into sections as follows:

 

    @media(min-width:320px){
        [ Some code ]
    }
    @media(min-width:400px){
        [ Some code ]
    }
    @media(min-width:768px){
        [ Some code ]
    }
    @media(min-width:850px){
        [ Some code ]
    }
    @media(min-width:992px){
        [ Some code ]
    }
    @media(min-width:1210px){
        [ Some code ]
    }

If you would like to edit styling of the specific device width, simply find the appropriate label in the responsive.css file, and then scroll down until you find the appropriate style that needs to be edited.

C) JavaScript - top

This theme imports following Javascript files.

  1. jquery-1.11.1.min.js - Javascript library
  2. bootstrap.min.js - Boostrap framework
  3. modernizr.custom.js - Modernizr jQuery plugin
  4. owl.carousel.min.js - OwlCarousel jQuery plugin
  5. jquery.validate.min.js - form validator that is used for contact page (contact form) and single page (comment form)
  6. jquery.form.js- allows you to upgrade HTML forms to use AJAX
  7. jquery.navgoco.min.js - for make toggle menu on mobile display
  8. superfish.js - for make hover menu on desktop display
  9. jquery.flexslider-min.js - Flexslider jQuery plugin
  10. jquery.sliderPro.min.js - Slider pro jQuery plugin
  11. fitvids.js - for fluid-width video
  12. jquery.bxslider.min.js - for bx slider
  13. custom.js - contains custom script

Except custom.js, all of the plugins are packed, so you don't need to manually edit anything in these files. The only necessary thing to know is how to call the method. You can refer to the website of the plugin author I listed in the "Sources and Credits" section to learn more about them.
